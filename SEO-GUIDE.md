# Toolboard SEO Notes

## Canonical domain
- `https://toolboard.ademisler.com`

## Automated SEO assets
Generated by `node scripts/sync-site-data.cjs`:
- `sitemap.xml`
- `robots.txt`
- `tools/*.html` (tool-specific indexable pages)
- `categories/*.html` (category landing pages)
- `_includes/home-jsonld.html` (homepage schema graph)
- `_includes/category-links.html` (indexable category links)
- `_includes/tool-links.html` (indexable tool links)

## Source of truth
- Extension manifest: `../extension/config/tools-manifest.json`
- English copy fallback: `../extension/_locales/en/messages.json`

## Update checklist
1. Update extension manifest/tools first.
2. Run sync script.
3. Commit generated files.
4. Deploy the website.
5. Submit `https://toolboard.ademisler.com/sitemap.xml` in Google Search Console.

## Structured data targets
- Home: `WebSite`, `SoftwareApplication`, `Organization`, `ItemList`
- Category pages: `CollectionPage`, `ItemList`, `FAQPage`, `BreadcrumbList`
- Tool pages: `SoftwareApplication`, `FAQPage`, `BreadcrumbList`
